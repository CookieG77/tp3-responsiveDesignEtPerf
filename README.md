

# Installation

Dans un premier temps pour installer ce projet il vous faudra executer cette commande dans le dossier d'installation désirer (néccesite `git` d'installer): 
```bash
git clone https://github.com/CookieG77/tp3-responsiveDesignEtPerf
```

Pour installer le nécessaire pour faire tourner le script python strimlit il vous faudra vous assurez que `streamlit` est installer, on peut s'en assurer avec la commande :
```bash
pip install -r .\requirements.txt
```
