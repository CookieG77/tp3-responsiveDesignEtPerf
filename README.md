# Landing Page - Portfolio Développeur

## Description du produit

Cette landing page représente un portfolio de développeur web (moi) avec une approche moderne et responsive. Le site met en avant :

- **Un appel à l'action clair** : "Construisons quelque chose ensemble"
- **Une promesse de valeur** : Solutions sur-mesure adaptées aux enjeux réels des clients
- **Présentation des services** : Applications sur mesure, optimisation des performances, expériences interactives, et conseil
- **Design responsive** : Approche mobile-first avec adaptation progressive au desktop
- **Performance optimisée** : HTML sémantique, CSS optimisé, images embarquées en base64

Le site est créé et structuré de manière à :

- **Prioriser le contenu essentiel** : Hero section visible immédiatement avec CTA et promesse
- **Optimiser pour mobile d'abord** : Navigation en menu hamburger, mise en page verticale adaptée aux petits écrans
- **Adapter progressivement au desktop** : Navigation horizontale, grilles multi-colonnes, footer enrichi
- **Maximiser les performances** : Images encodées en base64, CSS minimal et optimisé, HTML sémantique
- **Assurer l'accessibilité** : Structure sémantique claire, contraste de couleurs optimisé, navigation intuitive
- **Réduire le temps de chargement** : Zéro requêtes externes, ressources embarquées, code léger

Le projet utilise Streamlit pour servir le contenu HTML/CSS de manière interactive, permettant un déploiement facile et rapide.

## Comment lancer localement

### Prérequis
- Python 3.11 ou supérieur
- Git

### Installation

1. Cloner le dépôt :
```bash
git clone https://github.com/CookieG77/tp3-responsiveDesignEtPerf
cd tp3-responsiveDesignEtPerf
```

2. Installer les dépendances :
```bash
pip install -r requirements.txt
```

3. Lancer l'application :
```bash
streamlit run app.py
```

L'application sera accessible à l'adresse : `http://localhost:8501`
